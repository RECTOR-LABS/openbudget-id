## üîß Local Deployment Method

The current deployment uses **local deployment script** (`deploy.sh`) that builds on VPS from GitHub.

**One-Command Deployment:**
```bash
./scripts/deploy.sh
```

**Deployment Workflow:**
1. ‚úÖ Local commits pushed to GitHub
2. ‚úÖ VPS pulls latest code
3. ‚úÖ Docker builds image on VPS (with caching)
4. ‚úÖ Container restarted with new code

**Deployment Architecture:**
```
Developer Machine (./scripts/deploy.sh)
        ‚Üì
GitHub (code repository)
        ‚Üì
VPS Git Pull + Docker Build
        ‚Üì
DNS (openbudget.rectorspace.com)
        ‚Üì
Nginx (port 80/443) ‚Üí SSL (Let's Encrypt)
        ‚Üì
Reverse Proxy ‚Üí localhost:3100
        ‚Üì
Docker Container (openbudget-web)
        ‚Üì
PostgreSQL (172.18.0.1:5432)
        ‚Üì
Solana Devnet (Helius RPC)
```

**Deployment Speed:**
- First deployment: ~5-7 minutes (full build)
- Subsequent deployments: ~1-2 minutes (cached layers)
- Zero downtime with proper container restart

**Benefits:**

- ‚úÖ Build happens on production architecture
- ‚úÖ Secrets managed via gitignored .kamal/secrets
