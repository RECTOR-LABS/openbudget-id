# OpenBudget.ID Logo & Brand Assets

## Design Philosophy: Transparent Authority

The OpenBudget.ID visual identity embodies **Transparent Authority** - a design philosophy that communicates governmental accountability through geometric precision and technological clarity.

### Visual Language

**Hexagonal Network**: The interconnected hexagons represent:
- Blockchain's distributed ledger architecture
- Transparency through visible connections
- Institutional authority tempered by openness
- Verifiable pathways between data points

**Color Palette**:
- **Deep Blue (#0A2E5C)**: Institutional authority, trust, stability
- **Tech Blue (#2C7FB8)**: Technological verification, digital infrastructure
- **Vibrant Teal (#41B6E6)**: Real-time data, living transparency
- **Accent Orange (#D97D28)**: Human element, civic responsibility

## Usage Guidelines

### Web Applications

**Favicon**:
```html
<link rel="icon" type="image/png" sizes="32x32" href="/openbudget-icon-32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/openbudget-favicon-16.png">
```

**Apple Touch Icon**:
```html
<link rel="apple-touch-icon" sizes="512x512" href="/openbudget-icon-512.png">
```

**Open Graph (Social Media)**:
```html
<meta property="og:image" content="/openbudget-logo-square.png">
```

### Document Headers

Use `openbudget-logo.png` for:
- Slide decks and presentations
- PDF reports and documentation
- Email signatures (resize to 300-400px width)

### Social Media

- **Profile Pictures**: Use `openbudget-logo-square.png` or `openbudget-icon-512.png`
- **Cover Images**: Use `openbudget-logo.png` (crop/resize as needed)
- **Posts**: Use `openbudget-logo-square.png` for consistent branding

### Print Materials

For high-quality print:
- Use PNG versions for lossless quality
- Scale proportionally (maintain aspect ratio)
- Minimum readable size: 200px width for full logo

## Technical Specifications

**Resolution**: All assets are high-resolution (2x-3x standard display density)

**Color Mode**: RGB (for digital use only)

**Transparency**: PNG files have transparent backgrounds, JPG files have white backgrounds

**File Formats**:
- **PNG**: Lossless compression, supports transparency, best for web and digital use
- **JPG**: Lossy compression, smaller file size, best for photos and complex graphics

---

## Brand Consistency

### Do's ✓
- Maintain clear space around the logo (minimum 20% of logo height)
- Use provided color palette for brand consistency
- Scale proportionally (never stretch or distort)
- Place on white or very light backgrounds for maximum contrast

### Don'ts ✗
- Don't rotate or skew the logo
- Don't add effects (shadows, glows, gradients)
- Don't place on busy backgrounds that reduce legibility
- Don't change the color palette
- Don't separate the symbol from the text in the full logo
